# MERN Stack Course - Day 1 Assignment

## 📚 Overview
This repository contains my Day 1 practice assignment from the MERN Stack Course. The goal of this assignment was to learn the fundamentals of HTML and CSS by creating a simple webpage and applying basic styling.

## 🚀 Topics Covered

### HTML Basics
- HTML5 Document Structure
- DOCTYPE Declaration
- Headings (h1, h2, h3, h4)
- Paragraphs
- Navigation Buttons
- Lists

### CSS Basics
- External CSS File
- CSS Classes
- Background Colors
- Text Colors
- Button Styling
- Margins and Padding
- Basic Layout Design

## 🛠️ Project Features

- Created a complete HTML skeleton
- Added page heading and description
- Designed 8 navigation buttons
- Linked an external CSS file
- Applied custom button styles
- Created an IRCTC-themed sample webpage
- Organized files using proper folder structure
